# LazyVPS Airport Chain / 机场链与外购节点链路说明

## 一句话说明

机场链不是把机场订阅写进脚本。  
它是指用户在自己的客户端中导入外购机场订阅，然后把 AI / 流媒体域名规则指向对应机场策略组。

## 适合场景

```text
自建 VPS 普通上网体验好；
但某些 AI / 流媒体平台更依赖家宽、低风控、特定地区落地；
这时可以让 AI / 媒体域名走外购机场策略组。
```

## 典型链路

```text
普通网站 → 自建 VPS / 香港入口
GPT / Claude → 机场 AI 策略组
Netflix / Disney+ → 机场流媒体策略组
```

## 和 Media DNS 的区别

| 方式 | 作用 |
|---|---|
| Media DNS | 改善 DNS / CDN 区域解析，不改变出口 IP |
| 机场链 | 指定域名直接走外购节点，出口 IP 会变 |
| Server AI Routing | 在入口 VPS 服务端把 AI 域名转发到日本 / 台湾小鸡 |

## 开源安全原则

脚本不内置：

```text
机场订阅 URL
Token
节点 password
私有域名
```

只提供说明、规则思路和模板方向。
