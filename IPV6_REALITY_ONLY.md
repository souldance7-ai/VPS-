# LazyVPS v1.2.12 IPv6 Reality Only

## 用途

用于替代 Trojan over IPv6，解决 FLClash/Mihomo 环境下 IPv6 Trojan 持续 Timeout 的问题。

## 快速命令

```bash
bash /root/lazy-vps-menu.sh --quick ipv6-reality
```

## 输出

```text
01_IMPORT_FLCLASH_IPV6_REALITY_PORT2444.yaml
```

## 建议

优先使用 AAAA 域名，不要开 Cloudflare 代理小云朵，只用 DNS only。
