# GitHub 网页上传步骤

1. 解压 `LazyVPS-Interactive-v1.4.0.zip`。
2. 打开里面的 `LazyVPS-Interactive-v1.4.0` 文件夹。
3. Ctrl + A 全选里面所有内容。
4. 打开 GitHub 仓库：`Add file` → `Upload files`。
5. 拖入所有文件。
6. Commit message 写：

```text
feat: add LazyVPS interactive TUI v1.4.0
```

7. Commit 后点开根目录 `lazy-vps-menu.sh`，确认第一屏有：

```text
Formal Version: v1.4.0
Interactive TUI
```
