# LazyVPS Quick Menu Pack / 懒人建 VPS 快速菜单包

**正式版：v1.0 · 面板定稿版**  
**更新日期：2026-06-19**

## 界面定稿

- 顶部为彩色终端 ASCII 封面：LAZY VPS、太阳、沙滩、棕榈、躺椅懒人。
- 主菜单改为分区面板，不再使用会因中文字符宽度而错位的三列表格。
- `← / →` 切换分区，`↑ / ↓` 选择功能，`Enter` 执行，`1-32` 直达，`Q` 退出。
- 下方不使用渐层线，仅以固定颜色区分功能类别。

## 使用

```bash
chmod +x lazy-vps-menu.sh
bash lazy-vps-menu.sh
```

## 仅预览界面

```bash
bash lazy-vps-menu.sh --preview
```

## 分享安全

不内置个人 VPS IP、私有域名、节点密码或订阅地址。
